Итак, вы хотите улучшить ваши разговорные навыки.

Для этого вам нужно выбрать интересные вам диалоги на сайте и постараться их выучить или создать собственный диалог и сделать его доступным для всех. Далее нужно будет найти партнера для изучения диалога. Для этого есть два варианта позвоним самому или дождаться звонка.

Проявить инициативу и позвонить

Ваша роль в диалоге будет роль ученика.

Что бы позвонить кому нибудь перейдите на страницу "ДИАЛОГИ В ОЖИДАНИЕ ПАРТНЕРА".

Если кто нибудь из пользователей сервиса будет настроен на прием звонка вы увидете список из таких пользователей. У каждого пользователя будет или список диалогов или просто отображен skype id (если он введет его в своих настройках). Все что вам остается выбрать диалог и позвонить.

Ожидание звонка

Ваша роль в диалоге будет роль ведущего.

Первым делом вам нужно перейти в раздел "ВСЕ ДИАЛОГИ" и выбрать интересные вам диалоги.

Далее, вам нужно поставив главную галочку "принимать входящие звонки".

Теперь вам нужно дождаться, когда вам кто нибудь позвонит через сайт или если вы укажете ваш skype id в настройках, через скайп. В случае внешеного звонка вам нужно будет нажать на кнопку "Принять звонок"

Установка соединения.

При звонке система соединит двух участников диалога. Если будет невозможно установить голосовую связь, появится сообщение об этом. В этом случае у вас будет возможность текстового чата для установки контакта с партнером. Попыйтесь установить голосовую связь через другие программы, например, скайп.

Ваши действия после установления контакта.

После установки связи, партнеры делятся по ролям на ведущего (учитель) и ведомого (ученика): всегда - тот,кто звонит играет роль ведомого (ученика), а тот, кому звонят- ведущего(учителя). Диалоги состоят из шагов. На каждом шаге нужно произнести вслух свою фразу. На первом шаге сначала говорит ведущий. Потом ведущий переключается на партнера, и затем ведомый должен произнести свою фразу. После этого ведущий должен переключиться на следующий шаг диалога.

Возможные действия ведущего.

Как было сказано - от него зависит весь процесс прохождения диалога. Он руководит диалогом, переключаясь на партнера и переходит на следующие шаги диалога. Ведомый должен в свою очередь только произнести свою фразу.

Когда будет достигнут последний шаг диалога, ведущий может выбрать любой другой диалог (из тех что он заранее выбрал для принятия входящих звонков). Если он это сделает, оба партнера переключатся на другой диалог.

В случае если вы хотите выучить определенный диалог, а его нет в системе, вы можете его создать самостоятельно.

Перейдите на страницу "МОИ ДИАЛОГИ". Нажмите кнопку "Создать диалог". Появится окно с возможностью ввода наименования диалога. После ввода имени,нажмите кнопку "Создать". Далее будет создан диалог, и вы попадете в режим редактирования диалога, где нужно будет заполнить необходимые поля. Нужно заполнить описание диалога, выбрать требуемый уровень знания английского, желательно подобрать картинку диалога.

Далее самое главное, нужно написать сценарий диалога. Для это необходимо перейти на закладку "Сценарий диалога". Здесь редактор шагов. На каждом шаге можно выбрать первого персонажа, который будет начинать диалог. Далее выбрать персонаж и заполнить требуемые поля: задания для шага и правильную фразу (которую нужно будет выучить). Потом нажать "Добавить следующий шаг" для создания нового шага диалога. И таким образом создать все требуемые шаги.

В любой момент вы можете удалить ненужный шаг.

После того как все будет создано, можно сделать диалог доступным для всех. Для этого на закладке "Сделать доступным для всех" есть переключатель "Доступно для всех".Когда вы его выберите, диалог станет доступным для всех.

Вы так же можете удалить в любой момент созданный вами диалог, на закладке "Описание диалога", нажав на кнопку "Удалить диалог".
